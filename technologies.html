Used Technologies

As a data engineer, working with AWS technologies is part of my everyday routine, whether it involves Redshift, Aurora, S3, or other services. I initially taught myself AWS through personal projects, such as implementing a data pipeline. I gained hands-on experience with networking aspects like deploying applications, configuring inbound/outbound rules on EC2 instances, managing VPCs and subnets.

I first used Docker in 2020 at PwC when developing software for tax professionals. Since then I have studied containerization through "Docker Deep Dive" and learned to deploy multiple applications on a single server with proper isolation.

I adopted Podman for its security advantages, particularly rootless containers, while maintaining compatibility with Docker commands and Dockerfiles. I have a special interest in container networking, including name-based discovery within networks and IP-based communication across different networks and containerization technologies.

I use Apache Airflow at work to automate data pipelines and schedule ETL processes. I build DAGs to manage workflow dependencies and keep data pipelines running.

I build dashboards and data visualizations with Apache Superset and have experimented with its API to embed dashboards into other applications.

I use Python for everything - data pipelines, web backends, and dashboards. Built the Sankey Diagram Creator with Dash, developed Django REST APIs, and process data with pandas.

I have used other Database Management Systems such as MySQL, SQLite, Oracle, and MSSQL Server in the past. I have the most exposure to PostgreSQL (Redshift) and MySQL (Aurora) as they are commonly used in my work.

I was first introduced to Django at PwC as part of our development tech stack and researched its characteristics in a term paper, which led me to choose it as my primary framework for web application backends. In my Decoupled Django & React project, I used Django's backend capabilities and REST framework to create a web application with frontend-backend integration. In other projects implemented user registration and authentication.
